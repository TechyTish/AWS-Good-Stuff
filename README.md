# Custom Microsoft Teams Webhooks

Ever had a service where the webhook to notify a Microsoft Teams channel doesn't work?

Well this repository is dedicated to solving issues.

- Request Bin
- Postman
- AWS API Gateway
- AWS Lambda
- Microsoft Teams
